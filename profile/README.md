Connect orginal Gameboy hardware to the internet through a web browser.


Device Compatibility

* OEM Game Boy, Game Boy Color, and Game Boy Advance consoles
* Also compatible with FPGA consoles that support original link cable funtionality
    *  Analogue Pocket
    *  Chromatic
    *  Game Bub


**Online Multiplayer**

* Tetris, Dr. Mario, and Yoshi (GB)
    * Online 1v1 matches
    * Multiplayer battle royale–style games with 2 or more players
* Advance Wars 1 & 2 (GBA)
    * Play online multiplayer matches
    * Share and download custom maps
* Pokémon Generation III (GBA)
    * Native online battling
    * Trading
    * Record Mixing
* Pokémon Generations I, II & III
    * Online Pokémon trading.

**Game Boy Advance Utilities**

* Multiboot Loader
    * Send Game Boy Advance multiboot homebrew directly to a console.
    * No flash cartridge required.

**Game Boy Utilities**

* Game Boy Printer Emulation
    * Print directly to your computer.
    * Save Game Boy Camera photos as image files.
    * Supports any game with Game Boy Printer functionality.
* GB Cart Doctor
    * Backup and restore Game Boy and Game Boy Color cartridge ROMs and save files.
    * Requires an original Nintendo Game Boy Advance.

**Adapter Management**

* GB-Link Launcher
    * Update firmware directly from a web browser.
    * Configure adapter settings.
    * Access an up-to-date directory of compatible web applications.
